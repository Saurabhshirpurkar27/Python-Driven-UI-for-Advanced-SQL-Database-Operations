## ğŸ”§ How the Project Works

### Step 1: MySQL Database (Backend)

The database is designed to simulate real business systems.

**Database Components:**
- Tables for products, orders, inventory, and shipments
- Views for reports and summaries
- Stored Procedures for actions like order processing and stock updates
- Functions for business logic such as restocking checks

Business rules are enforced directly at the database level.

---

### Step 2: Streamlit Application (Frontend)

The Streamlit web app allows users to:

- View and filter data from tables and views
- Run stored procedures using buttons
- Add or update records using forms
- Execute database functions for calculations
- See real-time results without writing SQL
